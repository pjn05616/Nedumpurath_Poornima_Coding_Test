For this code to run properly, ensure that matplotlib, pandas, and openpyxl are installed.
Dowload the Nedumpurath_Poornima_USDACodingTest.py file and the Coding_Test.xlsx file (given from USDA) to the same directory. Using IDLE IDE (or a similar Python IDE), run the Nedumpurath_Poornima_USDACodingTest.py file. This will open two windows with the charts for the Oct sheet and the Nov sheet.
